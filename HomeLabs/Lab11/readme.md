## 📚 Теоретическая справка

### GRE (Generic Routing Encapsulation)
- **Туннельный протокол** для инкапсуляции пакетов в IP.
- Позволяет соединять **удалённые сети** через публичные сети.
- **Не обеспечивает шифрование** (используется с IPSec).
- Прост в настройке, но не масштабируется.

### DMVPN (Dynamic Multipoint VPN)
- **Улучшенная версия GRE** с динамическим установлением туннелей.
- **Hub-and-Spoke** архитектура с возможностью **Spoke-to-Spoke** связи.
- Использует **NHRP** (Next Hop Resolution Protocol) для поиска адресов.
- **Масштабируется** — новые Spoke добавляются без изменения Hub.
- Поддерживает **множество протоколов маршрутизации** (OSPF, EIGRP, BGP).

### Ключевые технологии DMVPN:
| Технология | Назначение |
|------------|------------|
| **NHRP** | Определяет NBMA-адреса (внешние IP) по туннельным адресам |
| **mGRE** | Один туннель для множества Spoke |
| **IPSec** | Опциональное шифрование (не используется в этой лабе) |
