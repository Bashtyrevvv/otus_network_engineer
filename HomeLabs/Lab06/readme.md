# Лабораторная работа №6 — BGP. Основы

## Цель
Настроить BGP между автономными системами. Организовать доступность между офисами Москва и С.-Петербург.

## Задание
1. eBGP Москва → Киторн (AS 1001 → AS 301)
2. eBGP Москва → Ламас (AS 1001 → AS 101)
3. eBGP Киторн ↔ Ламас (AS 301 ↔ AS 101)
4. eBGP Ламас ↔ Триада (AS 101 ↔ AS 520)
5. eBGP С.-Петербург → Триада (AS 2042 → AS 520)
6. IP доступность между Москвой и С.-Петербургом
