# 🌐 Лабораторная работа №9 — BGP для маршрутизации IPv6 unicast

## 🎯 Цель работы
Настроить **eBGP/iBGP IPv6 unicast** для всех сегментов сети по аналогичной логике с настройкой eBGP/iBGP IPv4 unicast.

---

## 📋 Задание

| № | Задача | Участники |
|---|--------|-----------|
| 1 | **eBGP IPv6** Москва → Киторн | R14 ↔ R22 |
| 2 | **eBGP IPv6** Москва → Ламас | R15 ↔ R21 |
| 3 | **eBGP IPv6** Киторн ↔ Ламас | R22 ↔ R21 |
| 4 | **eBGP IPv6** Ламас ↔ Триада | R21 ↔ R24 |
| 5 | **eBGP IPv6** С.-Петербург → Триада | R18 ↔ R24/R26 |
| 6 | **IPv6 связность** Москва ↔ СПб | ping |
| 7 | **iBGP IPv6** в Москве | R14 ↔ R15 |
| 8 | **iBGP IPv6** в Триаде (RR) | R23, R24, R25, R26 |

## 🔧 План настройки

### 1. Настройка IPv6 на интерфейсах

| Устройство | Интерфейс | IPv6 адрес | Назначение |
|------------|-----------|------------|------------|
| R14 | e0/2 | 2001:1001:78::1/64 | → Киторн |
| R15 | e0/2 | 2001:1001:77::5/64 | → Ламас |
| R21 | e0/0 | 2001:1001:77::6/64 | ← Москва |
| R21 | e0/1 | 2001:1001:77::1/64 | ↔ Киторн |
| R21 | e0/2 | 2001:1001:0::5/64 | ↔ Триада |
| R22 | e0/0 | 2001:1001:78::2/64 | ← Москва |
| R22 | e0/1 | 2001:1001:77::2/64 | ↔ Ламас |
| R24 | e0/0 | 2001:1001:0::6/64 | ↔ Ламас |
| R24 | e0/3 | 2001:1001:0::10/64 | → СПб |
| R26 | e0/3 | 2001:1001:0::22/64 | → СПб |
| R18 | e0/2 | 2001:1001:0::10/64 | → Триада |
| R18 | e0/3 | 2001:1001:0::22/64 | → Триада |

### 2. eBGP IPv6 сессии

| Соседи | IPv6 адреса |
|--------|-------------|
| R14 ↔ R22 | 2001:1001:78::1 ↔ 2001:1001:78::2 |
| R15 ↔ R21 | 2001:1001:77::5 ↔ 2001:1001:77::6 |
| R22 ↔ R21 | 2001:1001:77::2 ↔ 2001:1001:77::1 |
| R21 ↔ R24 | 2001:1001:0::5 ↔ 2001:1001:0::6 |
| R18 ↔ R24 | 2001:1001:0::10 ↔ 2001:1001:0::9 |
| R18 ↔ R26 | 2001:1001:0::22 ↔ 2001:1001:0::21 |

### 3. iBGP IPv6 сессии

| Соседи | IPv6 адреса |
|--------|-------------|
| R14 ↔ R15 | 2001:1001:77::254 ↔ 2001:1001:77::253 |
| R23 (RR) ↔ R24 | 2001:1001:0::251 ↔ 2001:1001:0::252 |
| R23 (RR) ↔ R25 | 2001:1001:0::251 ↔ 2001:1001:0::253 |
| R23 (RR) ↔ R26 | 2001:1001:0::251 ↔ 2001:1001:0::254 |
