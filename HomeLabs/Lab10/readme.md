# 🌐 Лабораторная работа №10 — Основные протоколы сети интернет

## 🎯 Цель работы
Настроить **DHCP**, **NTP** и **NAT** в офисах Москва, С.-Петербург и Чокурдах.

---

## 📋 Задание

| № | Задача | Что делаем |
|---|--------|------------|
| 1 | **NAT(PAT)** на R14 и R15 | Трансляция в адрес AS1001 |
| 2 | **NAT(PAT)** на R18 | Пул из 5 адресов AS2042 |
| 3 | **Статический NAT** для R20 | R15 → R20 |
| 4 | **NAT для R19** | Доступен для удалённого управления |
| 5 | **Статический NAT(PAT)** для Чокурдаха | R28 |
| 6 | **DHCP сервер** в Москве | VPC1 и VPC7 получают настройки по DHCP |
| 7 | **NTP сервер** в Москве | R12 и R13 синхронизируют время |
| 8 | **IP связность** всех офисов | Проверка ping между всеми сетями |

---

| № | Задача | Участники |
|---|--------|-----------|
| 1 | **NAT(PAT)** на R14 и R15 (AS 1001) | R14, R15 |
| 2 | **NAT(PAT)** на R18 (пул 5 адресов AS 2042) | R18 |
| 3 | **Статический NAT** для R20 | R15 ↔ R20 |
| 4 | **NAT для R19** (удалённое управление) | R14 ↔ R19 |
| 5 | **Статический NAT(PAT)** для Чокурдаха | R28 |
| 6 | **DHCP сервер** в Москве (VPC1, VPC7) | R12, R13 |
| 7 | **NTP сервер** в Москве | R12, R13 |
| 8 | IP связность всех офисов | Все сети |

| Внутренний IP | Внешний IP | Устройство |
